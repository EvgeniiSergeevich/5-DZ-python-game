Игра "117 конфет"

Правила:
На столе лежит 117 конфета. Играют два игрока делая ход друг после друга. Первый ход определяется жеребьёвкой. 
За один ход можно забрать не более чем 28 конфет. Все конфеты оппонента достаются сделавшему последний ход.
_______________________________________________________________________________________________________________

При запуске игры пользователь вводит количество игроков (1 или 2). Если введены другие числа или тип данных не int,
игра закроется, с сообщением об ошибке.
После выбора количества игроков, пользователь вводит своё имя (для двоих игроков - 2 имени по очереди. У бота есть имя)

Происходит автоматическая жеребъёвка.

Далее начинается игра, где пользователь вводит с клавиатуры количество конфет, которое он хочет забрать (от 1 до 28 штук)
Если количество конфет выходит за этот диапазон, вылетает сообщение об ошибке и предложение заново ввести количество конфет.
Так же, если ввести вместо типа int другой, так же вылетит сообщение об ошибке и предложение с вводом числа заново.

При вводе символа 'q', игра закрывается.

Так же, у бота есть своя логика, при подсчёте можно его победить, но, если не задумываясь забирать конфеты, то шансы
проиграть очень высоки!
